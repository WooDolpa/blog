## @Tag
`@Tag` 어노테이션은 API 엔드포인트를 그룹화하는 데 사용됩니다.
이를 통해 API 문서에서 논리적으로 관련된 엔드포인트들을 함께 묶어 더 이해하기 쉽게 만들 수 있습니다.

<table>
    <thead>
        <tr>
            <th>속성</th>
            <th>설명</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>name</td>
            <td>태그명</td>
        </tr>
        <tr>
            <td>description</td>
            <td>태그에 대한 설명을 제공합니다.</td>
        </tr>
    </tbody>
</table>

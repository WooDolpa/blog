# HTML/CSS 실무 커리큘럼 체크리스트 (10주)

> 사용법: 각 항목을 완료할 때 ✅로 표시하세요. 각 주차는 **핵심 → 실습 → DoD → 회고** 순서입니다.

---

## 공통 품질 게이트 (Definition of Done)

- [ ] 시맨틱 구조(landmarks: header/nav/main/footer)와 올바른 문서 outline 사용
- [ ] 키보드 전탐색 가능, `:focus-visible` 스타일 제공, 스크린리더 레이블 정확
- [ ] 디자인 토큰(CSS 변수)로 색/간격/타이포/레이어 통일, 테마 전환 가능
- [ ] 특이성 0~1단 유지(id/!important 금지, 유틸 레이어 예외), @layer 설계 적용
- [ ] 모바일 퍼스트 + 컨테이너 쿼리로 컴포넌트가 크기에 적응
- [ ] 이미지 `srcset/sizes`/포맷(WebP/AVIF) 적용, 폰트 최적화(`font-display: swap`)
- [ ] Lighthouse: 성능/접근성 ≥ 95, 스타일린트/프리티어 에러 0
- [ ] CSS 예산: gzip 기준 < 60KB(프로젝트 기준), 중복 규칙 제거
- [ ] CI에서 lint/format/build 자동화, 브라우저 지원 매트릭스 문서화
